## Key Features Overview

1. **File-based Routing** - Create routes by adding files to the `app` directory
2. **Server-side Rendering** - Built-in SSR for better SEO and performance
3. **Static Site Generation** - Pre-render pages at build time
4. **API Routes** - Build backend functionality alongside your frontend
5. **Image Optimization** - Automatic image optimization with the `next/image` component

## Getting Started

To create a new Next.js 14 project:

```bash
npx create-next-app@latest my-app
cd my-app
npm run dev
```

This will set up a new Next.js project with all the latest features and best practices.
