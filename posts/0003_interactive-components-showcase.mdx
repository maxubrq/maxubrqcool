## Code Playground

Try out some code directly in the blog:

<CodePlayground 
  initialCode={`// Try some JavaScript
const message = "Hello from MDX!";
console.log(message);

// Calculate fibonacci
function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}

console.log(fibonacci(10));`}
  language="javascript"
/>

## Conclusion

Interactive content transforms static blog posts into engaging, hands-on experiences. With MDX, you can:

- **Engage readers** with interactive elements
- **Test knowledge** with quizzes
- **Provide hands-on experience** with code playgrounds
- **Organize content** with tabs and progress indicators
- **Create memorable experiences** that stick with readers

The possibilities are endless when you combine the power of React components with the simplicity of markdown!
