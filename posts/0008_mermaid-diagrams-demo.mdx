# Mermaid Diagrams Demo

This post demonstrates the integration of Mermaid diagrams in your blog. Mermaid is a powerful tool for creating diagrams and flowcharts using simple text syntax.

## Flowchart Example

Here's a simple flowchart showing a decision process:

<Mermaid title="Decision Flow">
{`graph TD
    A[Start] --> B{Is it raining?}
    B -->|Yes| C[Take umbrella]
    B -->|No| D[Check weather app]
    C --> E[Go outside]

## Features

The Mermaid component includes several useful features:

- **Theme Integration**: Diagrams automatically adapt to your light/dark theme
- **Copy Code**: Copy the diagram source code to clipboard
- **Download SVG**: Download diagrams as SVG files
- **Refresh**: Re-render diagrams if needed
- **Error Handling**: Clear error messages for invalid syntax
- **Responsive**: Diagrams scale properly on all screen sizes

## Usage

To use Mermaid diagrams in your MDX files, simply wrap your diagram code in the `<Mermaid>` component:

```jsx
<Mermaid title="Your Diagram Title">
{`graph TD
    A[Start] --> B[End]`}
</Mermaid>
```

The component supports all Mermaid diagram types including flowcharts, sequence diagrams, Gantt charts, class diagrams, state diagrams, pie charts, and Git graphs.
