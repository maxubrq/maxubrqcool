## Quiz Features

### Question Types

1. **Single Choice**: Traditional multiple choice with one correct answer
2. **Multiple Choice**: Select multiple correct answers with partial credit
3. **True/False**: Simple binary questions
4. **Text Input**: Free-form text with regex pattern matching

### Scoring Modes

- **Practice Mode**: Instant feedback, hints, and explanations
- **Exam Mode**: Timed, with streak bonuses and time bonuses
